Remix Splitter

General Info

This app is a Python-based song stem separator that allows remixing using guitar-style effects

Tracks can be exported! Export an individual one just by soloing it, or export a remixed version with effects of chocie!

There is a ready-to-use build for Windows available

There were some issues exporting it to Mac as a standalone application. I should have a ready to go build up soon, but it can still work if you do the following:

1) Ensure you have Python version 3.10 specifically installed

2) On the Github page below, head over to the Mac branch and clone it

3) Open a Terminal to the folder in which you downloaded the code, and run "pip install -r requirements.txt"

4) Run the Python code in the main.py folder once you have all the modules from requirements.txt, and it should be ready to go!
