Remix Splitter

Alan Gill - 0analogspace@gmail.com

General Info

This app is a Python-based song stem separator that allows remixing using guitar-style effects. This was made as a final project for UCSD's ICAM Major in 2025

This app allows for the splitting of songs to seperated instruments, adding effects, changing volume, and then exporting them

Pre-compiled downloads coming soon!

Running from source code

There were some issues exporting it as a standalone application. I should have a ready to go build up soon, but it can still work if you do the following:

1) Ensure you have Python version 3.10 specifically installed

2) On the Github page below, download either:

If you are on Windows, you can download from the main branch

If you are on Mac, there is a dedicated Mac branch

3) Open a Terminal to the folder in which you downloaded the code, and run "pip install -r requirements.txt"

4) Run the Python code in the main.py folder once you have all the modules from requirements.txt, and it should be ready to go!
